# 网站标题的书写

- 字数26以内
- 符号尽量用-
- 放品牌词
- 核心词在中间
- 核心词-品牌词（俄罗斯旅游-鲟鱼酱旅行）
- 核心词-需求词-品牌词

# 网站描述

```html
<meta name="description"/>
```

- 描述需要包含标题
- 对排名影响不大

# 网站关键字

```html
<meta name="keywords"/>
```

- 英文逗号
- 从标题提取
- 不要堆积词语
