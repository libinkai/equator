# 模式切换

## VIM的模式

- 正常模式
- 插入模式
- 命令模式
- 可视模式

## 进入编辑模式

- ```
  - i 光标钱进入 insert
  - I 光标所在行行首进入
  - a 光标后边进入 append
  - A 行尾进入
  - o 光标下方插入空行并进入
  - O 光标上方插入空行并进入
  - s 删除光标所在字符并进入
  - S 删除光标所在行并进入
  ```

## 退出编辑模式

- ```
  - wq 保存并退出
  - q! 强制退出
  - w path 另存为
  ```

# 光标定位

## 以字符为单位移动

```
-h l 左右
-j k下上
- 可以搭配数字使用
```

## 快速移动

``` 
- 0 行首
- $ 行尾
- gg 文件首行
- G 文件尾行
- b 光标所在单词的头部
- e 光标所在单词的尾部
- w 光标所在的单词的下一个单词
- 行号+G 跳转到该行
- :+行号 跳转到该行
- % 快速定位到光标所在的括号的另一半括号
```

# 内容编辑

## 删除

> 删除实际上是剪切，数据被保存到缓冲区

- ```
  - x 删除单个字符
  - 删除多个字符  -d motion（范围）
  	-d0 或者 d^
  	-d$
  	-db
  	-de
  	-dw
  	-dh 删除光标前一个字符
  	-dl 删除光标所在字符
  	-dj 删除本行以及下一行
  	-dk 删除本行以及上一行
  	-dd 删除一行
  	-dgg
  	-dG
  ```

## 撤销

```
-u undo 上一次
-U undo 整行
-ctrl+r redo
```

## 粘贴

```
-p 粘贴在光标之后
-P 粘贴在光标之前
注意：如果之前剪切（删除）一整行，粘贴到小一行；否则，从光标后开始粘贴
```

## 拷贝

```
-y [数字] motion
```

## 替换

```
-r 替换单个字符
-[数字]r 
-R 进入替换模式
```

## 修改

> 删除并进入插入模式，修改=删除+进入插入模式

```
-c [数字] motion
```

## 缩进

```
>> 缩进
<< 反缩进
```

# 文件

- ctrl + g 查看文件信息

## 搜索

|          | / 目标         | ? 目标         |
| -------- | -------------- | -------------- |
| 搜索方向 | 光标向后       | 光标向前       |
| n        | 向后搜索下一个 | 向前搜索下一个 |
| N        | 向前搜索下一个 | 向后搜索下一个 |

- 取消高亮 `:nohl`
- `. * [ ] ^ % / ? ~ $`  10个字符需要用`\`转义

## 替换

```
:s/old/new 替换光标所在行第一个

:s/old/new/g 替换光标所在整行

:%s/old/new/g 替换整个文件

:%s/old/new/gc 替换整个文件并每次替换之前询问

:5,16s/old/new/g 替换5到13行
```

## 合并文件

```
- :r name 读取并置入光标的下一行
```

## 打开多个文件

```
- o 水平并排 
- O 垂直并排
- ctrl + w + w 切换下一个文件
- ctrl + w + 上、下 上下切换
- ctrl + w + 左、右 左右切换
- :qa :wqa :q! 一键退出多个文件 
```
